This game is a game just for bits of fun. It is a duel game that could be played between two player either on keyboard or online, also, this game has a premature AI system that do not know how to jump =( 

This is a game purely coded by one person and his loyal AI-Companion, but thanks to several friends who devoted efforts in designing the balance and creation of new elements the program is becoming more and more complex and enjoyable. 

This project started as a weekend boost idea personal project, but with soo many people playing and enjoying the game, I hope I could develop this game to recall everyone to that time when we play game physically side-by-side, with friends and families, purely out of fun.
